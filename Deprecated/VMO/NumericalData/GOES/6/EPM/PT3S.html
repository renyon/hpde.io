Direct access to the GOES 6 EPM data
URL
ftp://satdat.ngdc.noaa.gov/sem/goes/data/full
Description

Direct link to the GOES 6 EPM data in FITS format as well as the energetic particle data from other GOES spacecraft. The data are sorted by year. The file names indicates the data type, spacecraft number, year, month, and resolution. For example, P0137607.FIT indicates particle data (P) for spacecraft 1 (01) at 3 second resolution (3) from July 1976 (7607). Note that spacecraft IDs 91 and 92 refer to SMS satellites, not GOES.

Parameter #2
Name
Flag Word
ParameterKey
Flag Word
Description

The flag word contains packed information about the data such as the multiplex counter, data processing error flag, the satellite command state, and archive version number. The multiplex counter value indicates the 3 data channels from which the counts per seound are recorded. Because ths multiplex counter can change every 3 sec is is not possible to give the data channel, energy range, and other information in the next 3 parameters. To calculate the multiplex counter, multiplex counter = flagword mod 16. With the multiplex counter use the look up table at see https://satdat.ngdc.noaa.gov/sem/goes/software/part.txt to determine the energy channels, the associated particle, and the energy range.
