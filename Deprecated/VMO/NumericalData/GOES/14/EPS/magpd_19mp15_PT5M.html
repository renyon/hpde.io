Description

The GOES 14 Magnetospheric Proton Detector (MAGPD) are essentially the same as the Energetic Particle Sensor (EPS) on GOES 8-12. For more information, please see T. G. Onsager et al. (1996), Operational uses of the GOES energetic particle detectors, in GOES-8 and Beyond, Proc. SPIE, Vol. 2812, edited by E. R. Washwell, pp. 281-290, Bellingham, WA. These data file contain mainly information on the proton count rates and fluxes from 95-475 keV. Data files contain long descriptive header.

Description

Direct link to the GOES 14 Magnetospheric Proton Detector (MAGPD) data in CSV format as well as the energetic particle data from other GOES spacecraft. The data are sorted by year, month, spacecraft, and file type. The file names indicates the data type, spacecraft number, year, month, and resolution. for example, g14_magpd_19mp15_5m_20121001_20121031.csv, indicates goes 14 magnetometer data at 5 min resolution from start of Oct 1, 2012 to the end of Oct 31, 2012.
