Wind MFI Data

ResourceID
spase://VHO/NumericalData/Wind/MFI/PT03S

Description

Wind MFI composite data. The files contain multiple
time resolution data: 3-second, 1-minute, and 1-hour. Data come in
either version 3 or version 4 with version 4 preferred when available.
Both are calibrated science quality data, however, the version 4 data
has a small, final correction applied to the Bz component which is not
found in the version 3 data. Details of this correction can be found
in the caveats section of this document. The version 4 correction is
a manual correction for long term drifts which can not be applied upon
first release of the data. Thus, version 3 files are released to within
a few days of the present to allow quick access to the data.
Subsequently, the manual corrections are applied and version 3 files
are replaced with a corresponding version 4 file. Multi-resolution
data are available in CDF for ftp downloads from Wind and SPDF sites
and with subset, plot and list capabilities from Wind and CDAWeb.
Single resolution data (3s or 1m or 1h) are available in ASCII for
ftp downloads from SPDF.

Caveats
Version 3 and 4 data are both science quality data. However, version 4 files contain a final Bz correction not found in the version 3 files. The correction is on the order of 0.1 nT and is applied manually after release of the initial version 3 data. The generation of the version 4 files requires many months of data prior to and after the corrected time periods. Thus it will always significantly lag the release of the version 3 files. When version 4 files become available the corresponding version 3 file is removed and the version 4 file becomes the only file available for that time period
