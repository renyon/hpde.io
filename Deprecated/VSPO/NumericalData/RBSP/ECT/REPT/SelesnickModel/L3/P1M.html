RBSP Energetic Particle, Composition, and Thermal Plasma Suite (ECT) Relativistic Electron Proton Telescope (REPT) Selenick Radiation Belt Proton Intensity Model, Level 3 (L3), Monthly Data

ResourceID
spase://VSPO/NumericalData/RBSP/ECT/REPT/SelesnickModel/L3/P1M

Description

The Files associated with the Van Allen Probes Selenick Model Data Product contain Model Radiation Belt Proton Intensity derived from Data taken by the Van Allen Probes REPT Instruments using the Method described by: Selesnick, R.S., Baker, D.N., Kanekal, S.G., Hoxie, V.C., and X. Li, (2018). Modeling the Proton Radiation Belt with Van Allen Probes Relativistic Electron-Proton Telescope Data. J. Geophys. Res.: Space Phys., 123. https://doi.org/10.1002/2017JA024661. Each File contains average Intensity, as a Function of Kinetic Energy, Equatorial Pitch Angle, and L-Shell, derived from one Month of Data.
