Explorer 34 Triaxial Fluxgate Magnetometer data

ResourceID
spase://VSPO/NumericalData/Explorer34/Magnetometer/PT2.5S

Description

This dataset contains 2.5 second measurements of the magnetic field data from the triaxial
fluxgate magnetometer and position measurements from Explorer 34 spacecraft in several different coordinate
systems. These data were not well documented. Time is most likely in columns 25-27. Time is given
approximately every 20 s. For each time there are frames (packets of data) 1, 3, 5, 7, 9, 11, 13,
and 15. For each frame there are a set of magnetic or position vectors. Mostly likely a frame occurs
every ~2.5 s. Note that two columns appear to have been lost because the descriptions of the
columns are not clear. Note that some of the data files are corrupted. The stop time given for
those files is the available last good time.
